# Using AI Tools to Build S.E.R.A.

AI tools are allowed as development accelerators, not as unchecked architects.

## Good use

- architecture review
- scoped implementation
- test drafting
- code review
- bug diagnosis
- documentation drafts
- migration analysis

## Bad use

- broad unreviewed generation
- direct-to-main changes
- bypassing certs
- inventing package boundaries without review
- adding future-dream features before the secure base is stable

## Tool roles

- ChatGPT/Sage: strategy, architecture, review, planning.
- Codex: scoped implementation tasks with acceptance criteria.
- Copilot: local coding assistance.
- Replit: prototypes and experiments only.
- Base44: mockups/internal tool prototypes only.
- Ollama: optional future local LLM provider adapter.

Core law: AI can propose, but certs decide.
