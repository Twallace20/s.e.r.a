# Non-Negotiables

1. Do not call a capability working unless it has a passing cert.
2. Do not let generated runtime data become source code.
3. Do not allow tools to bypass the safety layer.
4. Do not allow AI output to define done.
5. Do not add future-dream certs to the active passing suite.
6. Do not make the kernel depend on a model provider.
7. Do not hide failures behind no-op fallbacks.
8. Do not modify files outside the active workspace without explicit approval.
9. Do not log secrets.
10. Do not migrate legacy code without a clean interface, test, and safety boundary.
