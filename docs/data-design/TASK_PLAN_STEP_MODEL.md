# Task, Plan, Step Model

## Task

A task is what the user asked S.E.R.A. to do.

## Plan

A plan is S.E.R.A.'s proposed execution structure.

## Step

A step is one bounded action within a plan.

## Tool event

A tool event records an actual action taken.

## Safety event

A safety event records an allow/block/approval decision.

## Final report

A final report is the human-readable truth summary for the run.
